## 🟡 Важные (High)

### 3. Тесты 2FA не изолированы
**Файл:** `app/tests/test_2fa.py`
**Проблема:** Интеграционные тесты требуют реальной БД
**Решение:** Добавить mock fixtures для isolation
**Приоритет:** High

### 4. Отсутствие rate limiting на 2FA endpoints
**Файл:** `app/api/endpoints/two_factor.py`
**Проблема:** Возможны brute-force атаки на TOTP коды
**Решение:** Добавить rate limiting (5 попыток/мин)
**Приоритет:** High

### 5. Backup коды не шифруются
**Файл:** `app/utils/two_factor.py` строка 95
**Проблема:** Хэширование SHA256 недостаточно для backup кодов
**Решение:** Использовать bcrypt или argon2
**Приоритет:** High

### 6. Парсеры не обновлялись
**Файл:** `app/parsers/*/parser.py`
**Проблема:** Структура сайтов могла измениться
**Решение:** Проверить актуальность парсеров, обновить селекторы
**Приоритет:** High
